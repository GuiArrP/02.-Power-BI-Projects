Esse é um dashboard simples onde os dados foram tratados e a ferramenta de Bookmark é utilizada.

Obs: O dataset utilizado está em anexo, portanto basta baixar o mesmo e indicar o caminho no relatório para conseguir utilizar as interações do DashBoard. Para esse caso o relatório está com um parâmetro chamado path, portanto tente manter o nome do arquivo o mesmo e alterar apenas esse parâmetro dando o endereço até onde está o arquivo apenas, não é necessário colocar o nome e extenção do arquivo no final.
